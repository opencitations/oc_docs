# Search

[search.opencitations.net](https://search.opencitations.net) is a web interface for looking up citation data in OpenCitations. Given an identifier (such as a DOI), you can find out which works have cited a paper, or which works appear in its reference list.

No login or registration required.

## How it works

The interface is built around a single search box. You enter an identifier, choose whether you want to see **Citations** or **References**, and click search.

- **Citations**: the list of works that have cited the paper you searched for
- **References**: the list of works that appear in the bibliography of the paper you searched for

Each result is displayed as a record with two sides: the citing work and the cited work. For each one, you can see the title, the identifiers, the venue, the publication date, and the authors.

## Step by step

1. Go to [search.opencitations.net](https://search.opencitations.net)
2. Type an identifier in the search box, for example `10.1162/qss_a_00023`
3. Choose **Citations** or **References** from the dropdown
4. Click the search icon

The results page shows the total number of records found and lets you navigate through them using pagination. You can change how many results are shown per page.

## Supported identifiers

You can search using any of the following. Enter the value directly, without prefixes.

| Type | Example |
|------|---------|
| DOI | `10.1162/qss_a_00023` |
| PubMed ID | `33817056` |
| OpenAlex ID | `W3106215946` |
